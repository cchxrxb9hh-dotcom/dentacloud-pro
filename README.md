# DentaCloud Pro | Clinic Management System

A high-performance, AI-integrated dental clinic management system designed for efficiency and clinical precision.

## Key Features

### 1. Clinical Records & AI
- **Smart Progress Notes**: Clinical notes with Gemini-powered summarization and patient-friendly translation.
- **Digital Patient Folder**: Sidebar-integrated asset management for X-rays, photos, and clinical documents.
- **Document Automation**: Instant generation of Medical Certificates, Time Slips, and Specialist Referrals.

### 2. Schedule Management
- **Multi-Branch Flow**: Consolidated and per-branch views of daily clinical schedules.
- **Dynamic Rescheduling**: Quick date and time slot selection for updating patient visits.
- **Check-in Workflow**: Live tracking of patient status from 'Arrived' to 'Completed'.

### 3. Hardware Integrations
- **Imaging Bridge**: Custom protocol handler (`nnt://`) for direct integration with NewTom NNT workstations.
- **MyKad Support**: Smart card reading via WebUSB for automated patient registration (Malaysian Region).
- **Biometric Attendance**: Fingerprint-based clock-in system for staff shift and payroll management.

### 4. Technical Architecture
- **Performance**: Zero-animation CSS override for instantaneous UI responses.
- **Security**: Role-based access control (Admin, Doctor, Assistant, Accountant) and automatic inactivity session timeout.
- **Scalability**: Multi-branch support with distinct legal entity branding per location.

## Development & Deployment
The application is built as a modern ES6 React module system. It uses Tailwind CSS for styling and Lucide for iconography. AI services are handled via the Google Gemini API.
